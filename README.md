HAY is an iOS instant messaging application that provides features:
- [x] Login page: user can input email & password, then click login button
- [ ] Chatroom page: user can see message avatar and name opponent user on toolbar
- [ ] Chatroom page: user can see message list item (content & date sent)
- [ ] Chatroom page: user can input text & send it with sent button 
- [ ] Log out button: user can logout and show Login Page
